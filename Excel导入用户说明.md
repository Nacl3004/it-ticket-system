# Excel批量导入用户说明

## 功能说明

系统支持通过Excel文件批量导入用户,方便管理员快速创建多个用户账号。

## Excel文件格式要求

### 文件格式
- 支持的文件格式：`.xlsx` 或 `.xls`
- 第一行必须是表头（会被跳过）
- 从第二行开始是实际的用户数据

### 列顺序和说明

| 列序号 | 列名 | 是否必填 | 说明 | 示例 |
|--------|------|----------|------|------|
| A | 用户名 | 是 | 用户登录账号，必须唯一 | zhangsan |
| B | 密码 | 否 | 用户密码，留空则默认为123456 | 123456 |
| C | 真实姓名 | 否 | 用户真实姓名，留空则使用用户名 | 张三 |
| D | 邮箱 | 否 | 用户邮箱地址 | zhangsan@example.com |
| E | 电话 | 否 | 用户联系电话 | 13800138000 |
| F | 角色 | 否 | 用户角色名称，必须与系统中的角色名称完全一致，留空则为"普通用户" | 普通用户 |
| G | 部门 | 否 | 用户所属部门 | 技术部 |

### 系统角色列表

系统中可用的角色名称（必须完全一致）：
- 超级管理员
- 管理员
- HR
- IT运维
- 普通用户


## 导入步骤

1. 准备Excel文件，按照上述格式填写用户数据
2. 登录系统，进入"用户管理"页面
3. 点击"批量导入"按钮
4. 选择准备好的Excel文件
5. 点击"开始导入"按钮
6. 等待导入完成，查看导入结果

## 导入结果说明

导入完成后，系统会显示：
- **成功导入数量**：成功创建的用户数量
- **失败数量**：导入失败的记录数量
- **失败原因**：每条失败记录的具体原因（如用户名已存在、角色不存在等）

## 注意事项

1. **用户名唯一性**：如果Excel中的用户名在系统中已存在，该条记录会导入失败
2. **角色名称准确性**：角色名称必须与系统中的角色名称完全一致（区分大小写）
3. **权限要求**：只有超级管理员、管理员、HR和IT运维角色可以导入用户
4. **数据验证**：导入前请仔细检查数据的准确性
5. **批量操作**：建议每次导入不超过1000条记录，以确保系统性能

## 常见错误及解决方法

### 错误1：用户名已存在
- **原因**：系统中已有相同用户名的用户
- **解决**：修改Excel中的用户名或删除该行

### 错误2：角色不存在
- **原因**：Excel中填写的角色名称与系统中的角色名称不匹配
- **解决**：检查角色名称拼写，确保与系统角色列表完全一致

### 错误3：文件解析失败
- **原因**：Excel文件格式不正确或文件损坏
- **解决**：检查文件格式，重新保存为.xlsx格式
